# التراث ✦ — صفحة هبوط الحاجة نوشة

## 🧶 نظرة عامة
صفحة هبوط احترافية وعاطفية لمشروع كروشيه يدوي مصري أصيل باسم **"التراث"** تحت شخصية **"الحاجة نوشة"** - سيدة مصرية بخبرة 40+ عاماً في فن الكروشيه. الصفحة مصممة كـ **معرض فني** لا متجر تقليدي، بهوية بصرية ملكية دافئة (أوف وايت، بيج، ذهبي)، ونبرة عامية مصرية "شيك".

---

## ✅ الأقسام المنفذة

### 1. 🎬 Hero Section (الافتتاحية)
- عنوان رئيسي بخط أميري فاخر
- نص ترحيبي بلهجة مصرية عامية دافئة
- خلفية صورة احترافية مع تأثير باراكس
- جسيمات ذهبية متطايرة (particle animation)
- زرا CTA رئيسي وثانوي
- شريط إحصائيات متحرك: 40+ سنة خبرة، 1200+ قطعة، 850+ عميلة، 100% شغل يدوي

### 2. 📖 Story Section (حكاية الحاجة نوشة)
- نص عاطفي بلهجة مصرية أصيلة
- صورة رئيسية + صورة دائرية كمميز
- شارة "40+ سنة خبرة" متوهجة
- 4 بطاقات قيم (الأصالة، الإتقان، الإبداع، الدفا)
- تايم لاين تفاعلي لرحلة الحاجة نوشة من 1985 للآن

### 3. 🖼️ Gallery Section (متحف التراث)
- صندوق معلومات تاريخ الكروشيه بتصميم داكن فاخر
- جريد معرض فني 3×2 مع featured item
- Hover overlays بتحكي قصة كل قطعة
- Tags توضيحية لكل قطعة

### 4. ⚡ Challenge Section (مفيش مستحيل)
- 3 بطاقات: صورة من النت / فكرة من خيالك / هدية خاصة
- وعد الحاجة نوشة بخط مميز
- منطقة رفع صور (Drag & Drop / Click to Upload)
- معاينة الصورة فور رفعها

### 5. 💡 Consultation Section (مساحة الخبرة)
- **غرزة الأسبوع**: عرض تفاعلي بمخطط بصري + 3 غرزات قابلة للتصفح
- **نصيحة الحاجة نوشة**: 7 نصائح يومية تتغير تلقائياً مع زر تنقل
- **FAQ التفاعلية**: 6 أسئلة شائعة بانيمايشن فتح/إغلاق

### 6. 📬 Contact Section (تواصل مع الحاجة)
- سيدبار بروابط واتساب / فيسبوك / تليفون (Placeholder جاهز للإضافة)
- وعد الرد في 24 ساعة
- **اختيار الخدمة**: 3 خيارات مميزة (طلب مخصوص / صورة ننفذها / نصيحة)
- فورم كامل: الاسم، التليفون، المحافظة، الرسالة
- رسالة نجاح ظهور تلقائي بعد الإرسال
